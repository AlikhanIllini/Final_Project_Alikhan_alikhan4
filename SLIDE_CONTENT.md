## SCREENSHOT GUIDE

### Screenshots You Need:
1. **Dashboard** - Full view with 5-6 cards visible
2. **Card Detail** - One card's full information
3. **Create Card Form** - Adding new stock
4. **Tag Management** - Color-coded tags
5. **Filters in Action** - Before/after filtering
6. **Price History Table** - Showing data

### How to Take Good Screenshots:
- Use full browser window (not just partial)
- Clean test data (no Lorem Ipsum)
- Hide browser toolbar if possible
- Good lighting/contrast
- Crop to relevant area
- Annotate with arrows if needed

### Tools:
- **Mac:** Cmd+Shift+4 (select area)
- **Windows:** Win+Shift+S
- **Chrome:** DevTools responsive mode for clean shots
- **Annotation:** Preview (Mac), Paint (Windows), or Skitch
